This is a straightforward program intended for monitoring up to 4 streams at once.

Simply open the program and add your URLs to the appropriate boxes.

This will allow you to manage things like chats/titles/thumbnails/settings/etc without having to open each site in a new browser window.

Requires .NET 8.0
